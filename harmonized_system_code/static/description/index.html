User guide

How to create a new HS Code?

  1. Go to Sales > Configuration > HS Codes
  2. To create a HS Code you need to specify the Code, a Description, a HS Group and a Product.

How to use HS Groups?

To create a new HS Group, go to Sales > Configuration > HS Groups or create one directly from the HS Codes.

HS Groups are made to group different HS Codes according to the countries specified on the group.
