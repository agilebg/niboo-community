Project Workflow - Version Management

Manage the step of your tasks when changing the state of your version

This module makes the link between the Project - Light Workflow module and the Project - Version Management module.

It automatically changes the step of your tasks when you put a version in production.

User guide

How to use this module?

  1. Go in Configuration / Version State / Task Step Configuration.
  2. Fill in the fields. When a version will be set at the state filled in, all tasks of the type filled in will automatically be changed to the step filled.
  3. This is especially usefull when you set different steps for the done stage (e.g: tasks on the staging version state before being passed to the production version state).
